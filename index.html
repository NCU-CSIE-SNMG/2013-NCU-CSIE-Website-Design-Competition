2013 中央資工網頁設計比賽

重要公告

請務必詳細閱讀比賽辦法！

報名方式

  1. 從GitHub（點此進入Git & GitHub教學）上面Fork我們的網站樣板。
  2. 製作網站的內容，包括HTML、CSS與JavaScript的部份。
  3. 網站製作完成之後，向我們的專案發起Pull request，並記下Pull request的編號。
  4. 到報名網頁填寫報名資料。
  5. 下載並且列印切結書，填寫完畢後繳交到國立中央大學工程五館A305機房，取得報名編號，至此完成作品的繳交與報名。

比賽辦法

設計主題

為中央資工設計一個美觀而實用的系網頁。隨著行動上網的流行，越來越多人使用移動設備瀏覽網站，為增加系網頁的使用的方便性，以「符合Responsive web design以及使用者與內容為主，簡潔美觀而不過度裝飾與過多效果」作為新系網頁的設計主題。

作品規格

  1. 以內容呈現為主，避免過度設計的裝飾與特效。
  2. 利用我們提供的網站架構與內容（包含HTML與CSS檔案範例）進行創作，使用HTML、CSS與JavaScript設計，可以自由修改我們提供的網頁結構（HTML結構）與樣式。另外網頁需正常運作於Internet Explorer 9+、Google Chrome 20+、Firefox 4+、Opera 12+、Safari on Mac、Safari on iOS、Android等主要瀏覽器。
  3. 作品內容包括完成的HTML文件、CSS樣式表、使用到的JavaScript腳本以及網頁截圖（至少需要包含首頁的截圖），並將截圖以snapshot01、snapshot02等方式命名，格式以JPG與PNG為主。

競賽時間

自公布起至2013年06月16日為止（以信件寄出時間為準）

比賽獎勵

  1. 第一名可獲得15,000元整禮券。
  2. 第二名可獲得10,000元整禮券。
  3. 第三名可獲得6,000元整禮券。
  4. 佳作若干者可各得3,000元整禮券。

參賽規則

  1. 參賽者不限個人或團體，若為團體參賽，請在寄送作品時附上所有成員的資料，且比賽獎勵由該團體獲得並自行分配。
  2. 參賽者必須保證作品為原始創作，未侵害任何第三人之智慧財產權，並同意中央資工與中央資工網管小組能夠保存、修改並公開使用於中央資工網頁上。
  3. 中央資工網管小組收到作品之後會回信確認收到參賽作品。

評選方式

評選採用混合制評選，由中央資工網管小組初選符合要求者參加複選。複選採投票制，將於中央大學工程五館開放投票，並且以最高得票數者為最終得獎作品。

初選評選標準如下：

  1. 是否符合Responsive web design，能夠讓不同解析度的裝置都能夠瀏覽網站的重要內容，並且可以盡量多的瀏覽器正常運作。
  2. 使用者體驗是否良好，內容呈現是否清晰與易於瀏覽閱讀。
  3. 整體設計美觀與整齊統一，與資工系形象符合為佳，並參酌其他創意、細緻程度與相關設計良好部分。

其他事項

  1. 參賽作品應為原始創作並且僅限參加此次競賽，凡涉有抄襲或侵害他人著作權者，除得以取消參賽與得獎資格、追回獎金外，一切法律責任概由參賽者自行負責，並應賠償主辦單位之損失。
  2. 參賽作品不得有毀謗他人，發人隱私或其他侵害他人權利或違法之內容。如有違反法律規定，參賽者應自行負責。主辦單位並將依情節輕重，採取必要之措施，包括取消參賽與得獎資格，追回獎金等。
  3. 參賽作品無論錄取與否均不退件，請參賽者自留檔案。
  4. 得獎者將個別通知，並公告；未得獎者將不另外通知。
  5. 參賽者需服從評審之決定。如有發生糾紛，並應服從主辦單位所為之裁決。
  6. 主辦單位若發現參賽作品有違反本比賽規則所列之規定者，參賽者須自行負責且主辦單位得取消其參賽資格。若為得獎作品，則追回獎金並公告之。如造成第三者之權益損失，參賽者須自行承擔，不得異議。
  7. 凡報名參賽者，即視同承認本報名規定的各項內容及規定，本活動參賽規則如有異動，依最新消息公布為準。主辦單位保有所有相關活動最終解釋權與活動更改之權利。
